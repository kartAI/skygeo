# SkogRessurskart 16x16 (SR16)

[Bonitet]([url](https://www.nibio.no/tema/jord/arealressurser/ar50/skogbonitet)):
skogarealets evne til å produsere trevirke

- FGDB (6,3 GB) til FlatGeoBuf (13,3 GB), 30m 23sek:
```sh
  ogr2ogr -f FlatGeobuf 0000_4258_sr16_fgb 0000_4258_sr16_gdb.gdb sr16
```

## Tilgjengelighet og metadata
```sh
ogrinfo https://karttest.nibio.no/skygeo/sr16/bonitet_3857_cog.tif
ogrinfo -so https://karttest.nibio.no/skygeo/sr16/bonitet_3857_cog.tif sr16
```
