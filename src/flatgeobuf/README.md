# FlatGeobuf

## Format/teknologi
FlatGeobuf (FGB) er et åpent, effektivt og utviklervennlig vektorformat for geodata. Støttes i Python, JavaScript og moderne GIS-verktøy.

## Formål
Konvertere N50 vektordata fra GDB-format til FlatGeobuf for enklere deling, versjonering og bruk i webmiljøer.

## Getting started
- Installer avhengigheter: `pip install geopandas fiona pyogrio folium matplotlib mapclassify`
- Åpne `flatgeobuf_converter.ipynb` for stegvis konvertering fra GDB til FGB.

## Struktur
- `flatgeobuf_converter.ipynb`: Notebook for konvertering og demo
- `requirements.txt`: Python-avhengigheter
