# skygeo-workshop

Workshop-notebooks for Cloud Native Geospatial med Python og DuckDB.

## Notebooks

### [Cumulus_Geographica.ipynb](Cumulus_Geographica.ipynb)
Introduksjon til Cloud Native Geospatial i praksis. Du lærer å spørre mot store datasett i skyen med DuckDB, gjøre romlig analyse med GeoPandas, og visualisere resultater med Folium og lonboard. Bruker Overture Maps og norske kommunedata som eksempler.

### [gran_canaria_accommodation_density.ipynb](gran_canaria_accommodation_density.ipynb)
Analyserer tetthet av overnattingssteder på Gran Canaria ved hjelp av H3-heksagoner. Henter data fra Overture Maps, aggregerer per celle og visualiserer med interaktive kart.
